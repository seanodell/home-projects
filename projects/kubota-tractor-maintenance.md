## Description

Recommission the Kubota BX23S tractor package after sitting outdoors for 2-3 years in Redmond, WA. Extended outdoor exposure in the Pacific Northwest (heavy rain, some snow, summer heat) means aggressive attention to fluids, corrosion, rodent damage, and weathered rubber components.

## Phase 1 — Inspection

Before any fluids or start attempt.

- [ ] Walk-around: check for rodent nests (air filter, under hood, wiring), chewed hoses, rust, wasp nests in exhaust
- [ ] Tire pressures + condition (dry rot, cracking). Front 18x8.50-10: 17-22 psi. Rear 26x12.00-12: 14-17 psi.
- [ ] Open hood. Inspect fan belt for cracks (spec: 7-9mm deflection).
- [ ] Inspect all rubber hoses: radiator, fuel, power steering, hydraulic, intake air, breather. Look for cracks, swelling, chew marks.
- [ ] Remove air filter cover. Check housing for rodent debris. Install new outer (K1211-82320) and inner (K1211-82330).
- [ ] Inspect wiring and battery cables for rodent damage or corrosion.
- [ ] Note any damage needing parts. Stop if major wiring damage found.

## Phase 2 — Battery and Electrical

- [ ] Remove battery (ground - first, then +). Test voltage. Below 11.8V = replace.
- [ ] Install new battery (Group 26R, 540+ CCA). Connect + first, then ground -.
- [ ] Clean terminals with brush. Apply CRC protector spray.
- [ ] Key ON (don't start). Check all fuses (12 slots + 50A slow blow). Replace blown.
- [ ] Check all lights: headlights, hazards, turn signals, tail lights.

## Phase 3 — Fluids

Do NOT start engine. Bring drain pan, siphon pump, funnels, filter wrench, 17mm + 22mm sockets.

- [ ] Engine oil: drain (17mm plug, bottom of block), remove old filter (65mm wrench). Install new filter (HH150-32430), hand-tighten 1/2 turn past contact. New crush washer on plug. Fill 3.3L SAE 10W-30. Check dipstick.
- [ ] Fuel: siphon old diesel into jerry cans. Replace fuel filter (12581-43010). Fill ~7 gal fresh No. 2. Add Sta-Bil.
- [ ] Transmission: check dipstick (left side). If dark/burnt/2+ yrs old, full change — drain (22mm plug), clean strainer, refill 11.3L Super UDT-2. Otherwise top up.
- [ ] Front axle: check dipstick. If low/dark, drain both plugs, refill 3.6L Super UDT-2 or 80W-90. Reinstall breather plugs.
- [ ] Coolant: check recovery tank (H to L marks). If low/discolored, flush system, refill 3.1L 50/50 Zerex G-05 + distilled water.

## Phase 4 — Grease and Torque

Bring grease gun (EP2 + Moly EP cartridges) and torque wrench.

- [ ] EP2 grease: speed control pedal (pump till overflow), battery terminals, hood lock, hood guide
- [ ] Moly EP: all loader fittings — boom, bucket cylinder, linkage. Pump till fresh grease purges.
- [ ] Moly EP: all backhoe fittings — boom, dipperstick, bucket, swing, stabilizers. Pump till purge.
- [ ] Brake pedal free travel: 25-35mm
- [ ] Wheel bolts: front 110-132 lbf·ft, rear 80-96 lbf·ft
- [ ] Loader frame bolts: M14 at 108 lbf·ft, main frame at 150 lbf·ft
- [ ] Backhoe sub-frame bolts: 108 lbf·ft

## Phase 5 — First Start

- [ ] Bleed fuel: key ON ~30 sec (don't start). Fuel pump primes.
- [ ] Seat, parking brake, PTO off, range neutral, speed pedal neutral, loader locked.
- [ ] Accelerator to ~50%.
- [ ] Key ON — check Easy Checker lights. Coolant temp + low fuel should flash, gauges sweep.
- [ ] PREHEAT 5-10 sec.
- [ ] START. Max 30 sec crank. No start = key off, wait 30 sec, retry.
- [ ] Idle 5+ min. Oil pressure light MUST go off. Watch coolant temp. Check for leaks/noises.
- [ ] Exhaust: white/blue = normal cold start. Black = air filter or fuel issue, shut down.

## Phase 6 — Function Testing

Engine at ~1800 rpm. Move slowly. Watch for leaks.

- [ ] Loader: raise/lower boom, dump/rollback bucket, float position. Check for jerky response or leaks.
- [ ] Backhoe: rotate seat. Boom up/down, swing L/R, dipperstick in/out, bucket curl/dump, stabilizers. Watch for leaks.
- [ ] 3-point hitch: raise and lower.
- [ ] PTO: select rear, engage clutch. Shaft should spin. Disengage. Test mid PTO.
- [ ] Brakes: low range forward, apply brake. Should stop straight.
- [ ] 4WD: stop, engage front wheel drive, short drive, disengage.
- [ ] Drive low range forward + reverse. Check steering.
- [ ] Shut down. Re-check all fluid levels. Top up as needed.

## Notes

- **Rodent risk is high** — Redmond WA + sitting outdoors 2-3 years = very likely mice or rats nested somewhere. Wiring damage is the biggest concern (expensive to fix). Inspect thoroughly before attempting start.
- **Fuel is the #1 concern** — diesel that old will have water condensation and microbial growth. Drain completely, don't try to "use up" the old fuel.
- **Battery is almost certainly dead** — plan on buying a new one (12V, 560 CCA). Jump starting a deeply discharged battery can damage it.
- **The manual's "Removing the Tractor from Storage" section (p.146)** covers the basic procedure. This plan goes further because of the extended outdoor exposure.
- **Service interval reset** — after recommissioning, treat it as a "first 50 hour" break-in for service interval tracking purposes.
